# Sleep Debt Calculator

## About

This project calculates whether you are getting enough sleep in a week using JavaScript functions.

## What It Does

- Stores sleep hours for each day
- Calculates total actual sleep
- Calculates ideal sleep
- Compares both values
- Shows if you are sleeping enough or not

## Rules

- If actual sleep = ideal sleep → Perfect sleep
- If actual > ideal → Too much sleep
- If actual < ideal → Sleep debt

## Technology

- JavaScript (Functions, Conditionals)

## How It Works

1. Each day’s sleep is stored in a function
2. Weekly sleep is calculated
3. Ideal sleep is set (8 hours per day)
4. Both values are compared
5. Result is printed in console

## How to Run

1. Save file as `sleep-debt.js`
2. Open terminal
3. Run:

node sleep-debt.js
